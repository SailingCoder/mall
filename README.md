## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for test and launch server
$ npm run build
$ npm start:test

# build for production and launch server
$ npm run build
$ npm start:prod

# generate static project
$ npm run generate

```
